# Roadmap

## In progress

- **New version available:** Show in the TUI when a new version of flop7 is available.

## Planned

- **CLI tooling for simulations:** Allow users to run simulations and get results from the command line, without needing to use the TUI.
- **Improved simulation interface:** Improve the results pane, maybe with some graphs or at least a better progress bar.
- **Add logging for simulations:** At user request, save log files of events in Flip 7 games for future analysis or comparison.
- **Allow config files for simulations:** Allow users to set up a JSON configuration file for a simulation, then run simulations from it. This will likely be tied to the CLI tooling mentioned above.
- **Allow custom selection of trackers:** Allow users to select which trackers they want to run during a simulation from a list of all available trackers.
- **Improve command bar user experience:** Improve some clunky parts of the command bar, such as target selection during gameplay or bot configuration during simulation setup.

## Ideas

- **LAN games:** Give users the ability to play with each other over a LAN.
