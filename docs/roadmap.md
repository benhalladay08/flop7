## Planned

- **CLI tooling for simulations:** Allow the user to run simulations and get result from the command line, without needing to use the TUI.
- **Improved simulation interface:** Improve the results pane, maybe with some graphs or at least a better progress bar.
- **Add logging for simulations:** Per users request, save log files of events in Flip 7 games for future analysis or comparison.
- **Allow config files for simulations:** Allow user to set up a json file of configurations for a simulation, then run from that. Likely will be tied to the CLI tooling mentioned above.
- **Allow custom selection of trackers**: Allow the user to select which trackers they want to run during a simulation, from a list of all available trackers.
- **Improve command bar user experience**: Improve some clunky parts of the command bar, such as target selection during gameplay or bot configuration during simulation setup.

## Ideas

- **LAN games:** Give users the ability to play with each other over LAN.
